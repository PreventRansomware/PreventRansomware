# PreventRansomware.io
## Free,  practical protection against Ransomware.
Prevent Ransomware is an open source effort to beat ransomware via education. The documentation here aims to be understandable, concise and free from scaremongering. No cyber security expertise is required to implement it. No sponsorships, advertisements or commercial influence from cyber security vendors will be found. All content is peer reviewed and crowd sourced organically from the cyber security community.
